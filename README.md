# DeepSeek Usage Monitor

在 VS Code 状态栏实时显示 DeepSeek
API 的**账户余额**和**今日已使用金额**，帮助你随时掌控 API 消费。

## 功能

- 📊 **状态栏实时显示**：余额和今日用量一目了然
- 🔄 **自动轮询**：按设定间隔定时刷新，无需手动操作
- 💡 **智能暂停**：VS
  Code 失焦或最小化时自动停止轮询，聚焦时智能恢复（补齐剩余等待时间，不会多查）
- 👆 **手动刷新**：点击状态栏或执行命令即可立即更新
- 💾 **用量计算**：通过余额差值精确计算当日消费，数据跨会话持久化

## 配置

| 设置项                          | 类型   | 默认值 | 说明                                        |
| ------------------------------- | ------ | ------ | ------------------------------------------- |
| `deepseekUsage.apiKey`          | string | `""`   | DeepSeek API Key（以 `sk-` 开头）           |
| `deepseekUsage.refreshInterval` | number | `5`    | 自动刷新间隔（分钟），设为 `0` 禁用自动刷新 |

## 使用方式

1. 在 VS Code 设置中搜索 `deepseekUsage`，填入你的 API Key
2. 状态栏右侧会显示 `DeepSeek: ¥余额 | 今日 ¥用量`
3. 点击状态栏可手动刷新

### 命令行

- `DeepSeek Usage: Refresh` — 立即刷新余额和用量

## 今日用量计算原理

由于 DeepSeek 目前未公开用量查询 API，本插件通过**余额差值法**计算：

> 今日用量 = 当日首次查询时的起始余额 − 当前余额

每天首次刷新时自动记录起始余额，后续每次刷新实时计算差值。数据存储在 VS
Code 本地，重启后依然有效。

## 要求

- VS Code `^1.118.0`
- 有效的 [DeepSeek API Key](https://platform.deepseek.com/)

## 已知问题

- 如果在同一天内充值，余额增加会导致今日用量显示异常（变为负数时会被修正为
  `¥0.00`）
